# Reto-6


###  reto_6_1.py
Realiza operaciones aritméticas básicas (**+**, **-**, **\***, **/**).  
Valida los tipos de datos, el operador y evita la división por cero.

###  reto_6_2.py
Verifica si una palabra es un **palíndromo sin usar slicing**.  
Utiliza dos punteros (uno desde el inicio y otro desde el final) para comparar los caracteres.

###  reto_6_3.py
Filtra los números primos dentro de una lista.  
Implementa la función `is_prime()` y valida que todos los elementos sean enteros.

###  reto_6_4.py
Encuentra la **mayor suma entre dos números consecutivos** en una lista.  
Verifica que haya al menos dos elementos antes de realizar el cálculo.

###  reto_6_5.py
Filtra los **anagramas** utilizando una clave canónica (letras ordenadas).  
Devuelve solo las palabras que tengan al menos un anagrama dentro de la lista.

###  shape.py
Demostración del paquete `Shape` con validaciones y excepciones.  
Incluye las clases `Circle`, `Rectangle` y `Triangle`, con control de errores y escalado seguro.
